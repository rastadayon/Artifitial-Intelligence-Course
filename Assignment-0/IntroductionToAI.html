The dataset consists of the following columns and each column or feature has the specified type:

  • The student's serial number. This feature has integer values.
  • GRE score of the student. This feature has integer values between 0 to 340.
  • TOEFL score of the student. This feature has integer values between 0 to 120.
  • Ranking of the university in which the student was previously studying in. This feature has integer values between 0 to 5.
  • SOP (statement of purpose) score of the student. This feature is a floating point value ranging from 0 to 5.
  • LOR (letter of recmmendation) score of the student. This feature is a floating point value ranging from 0 to 5.
  • CGPA score of the student. This feature is a floating point value ranging from 0 to 10.
  • Research experience. This feature has a binary value (either 0 or 1).
  • Chance of the student getting addmitted. This feature represents the probability that the student will get admitted and has floating point values ranging from 0 to 1 (as they are probability values). The rows in this feature that have Nan values represent the test set.

Verbose: (bool, optional argument) This argument indicates whether to print the full summary or not. Default value follows the display.max_info_columns setting. True or False overrides the display.max_info_columns setting.

Chosing the variable with the largest correlation with Chance of Admit variable¶

To find which of the variablesin the dataframe have the largest linear correlation with Chance of Admit variable we can have a good guess by plotting them however to make sure, we calculate the correlation value. Correlation is a statistical measure of linear correlation between two variable. There are a few different mothods for calculating the correlation, one of them is the Pearson correlation coefficient. The Pearson correlation coefficient is calculated with this formula:

\begin{align*} \rho_{X,Y} = \frac{cov(X,Y)}{\sigma_X \sigma_Y} \end{align*}

In which the $cov(X,Y)$ is the covariance of the two variables. The $\sigma_X$ and $\sigma_y$ are standard deviations of X and Y variables and $\rho_{X,Y}$ represents the correlation value.

The coefficient value falls somewhere between -1 and +1 where a +1 indicates a perfect positive relationship and -1 indicates a perfect negative relationship.

Pandas library has a corr() function which when called without arguments on a data frame calculates the pairwise correlation of the columns (excluding NaN values) and returns a data frame.

The returned data frame from corr function will have 1 along the main diagonals and it is also symmetric.

Hypothesis function¶

This function given the theta values and input variables (X) produces an estimation for the output.

The output will be like the following: \begin{align*} h_\theta(x) = \theta_0 + \theta_1 x \end{align*} $\theta_0$ and $\theta_1$ are given as inputs to the function. $\theta_1$ is the slope for the linear relation and $\theta_0$ is the y-intercept.

In [98]:
def hypothesisFunc(theta0, theta1, X):
    return X* theta1 + theta0

When calling this method, the X input is a vector so when multiplied by theta1 every element will be multiplied and when added with theta0 the same applies. The theta values could have also been vectors however for this problem they were considered as numeric values.

Cost function¶

To check if the estimation is a good one we should find how far away it is from the actual value. To determine this cost we use a cost function. This cost function calculates Mean Square Error which is obtained from the following equation: \begin{align*} J_(\theta_0,\theta_1) = \frac{1}{2m} \sum_{i=1}^{m} (h_\theta(x^{(i)} - y^{(i)})^2 \end{align*}

In [136]:
def costFunc(theta0, theta1 , X ,y ):
    m = len(y)
    
    predictions = hypothesisFunc(theta0,theta1,X)
    cost = ((predictions-y)**2).sum()/(2*m)
    return cost

X is the vector of input values (CGPA scores) and y is the target value (Chance of Admit), theta0 and theta1 are same as the last part. Also m is the length of y vector (same as the number of train datas).

Generating theta values¶

The method used to find the best values for $\theta_0$ and $\theta_1$ here is to find the values that make the cost the least possible value. For this porpuse what we need is to do two partial derivations one for each variable. From the two resaulted equations the 2 variables can be obtained.

These are a summary of the mathematics used for calculating theta0 and theta1:

$$ SSE = \sum\limits_{i=1}^{n} {(Y_i - \hat Y_i)^2} $$$$ SSE = n( \bar Y - \theta_0 - \theta_1 \bar X)^2 + \left ( \theta_1 - { \sum\limits_{i=1}^{n} { (X_i - \bar X)(Y_i - \bar Y) } \over \sum\limits_{i=1}^{n} {(X_i - \bar X)} } \right ) ^2 + \sum\limits_{i=1}^{n} {(Y_i - \bar Y)^2} \left ( 1 - \left [ { \sum\limits_{i=1}^{n} {(X_i - \bar X)(Y_i - \bar Y)} \over { \sqrt { \sum\limits_{i=1}^{n} {(X_i - \bar X)^2} \sum\limits_{i=1}^{n} {(Y_i - \bar Y)^2} } } } \right ] ^ 2 \right ) $$$$ n ( \bar Y - \theta_0 - \theta_1 \bar X ) ^ 2 = 0 $$$$ \left ( \theta_1 - { \sum\limits_{i=1}^{n} {(X_i - \bar X)(Y_i - \bar Y) } \over \sum\limits_{i=1}^{n} {(X_i - \bar X)^2 } } \right ) ^ 2 = 0 $$$$ \hat \theta_1 = { \sum\limits_{i=1}^{n} {(X_i - \bar X)(Y_i - \bar Y) } \over \sum\limits_{i=1}^{n} {(X_i - \bar X)^2 } } $$$$ \hat \theta_0 = \bar Y - \hat \theta_1 \bar X $$

The specific mathematics of how these two variables are calculated can be found in this link

def estimationFunc(X, y):
    theta1 = ((X - X.mean())*(y-y.mean())).sum() / ((X-X.mean())**2).sum()
    theta0 = y.mean() - theta1*(X.mean())
    return(theta0, theta1)

This function inputs X and y vectors and outputs the theta values. The theta values are obtained from the equations that were talked about in the previous sections.
